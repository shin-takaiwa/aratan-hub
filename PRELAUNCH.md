# 公開前チェック（このアプリ用）

土曜の公開前にすべて消化する。原典は基盤リポの `docs/checklist-launch.md`。

- [ ] RLS 有効化（全テーブル ON・anon で他人のデータが読めない）
- [ ] 本番環境変数を Vercel Production に設定
- [ ] service_role 等の秘密がクライアントに露出していない
- [ ] エラー監視が動く
- [ ] 利用計測を入れた（アプリDBに溜めない）
- [ ] OGP画像・タイトル・説明（URL を貼って崩れない）
- [ ] モバイル表示で主要導線が破綻しない
- [ ] favicon / タブのタイトル
- [ ] 最低限のプライバシー表記
- [ ] 404 / エラー時に白画面化しない
- [ ] ハブにリンク＋メタを登録した
